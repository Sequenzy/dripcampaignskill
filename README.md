# Drip Campaign Skill

Static landing page and agent skill for dripcampaignskill.com. Works with Codex, Hermes, OpenClaw, Claude Code, Cowork, and other skill-aware agents.

Install shortcut:

```bash
npx skills add dripcampaignskill
npx skills add Sequenzy/dripcampaignskill --skill dripcampaignskill
```

Raw skill folder: `skills/dripcampaignskill`
